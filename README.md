# ats-codesign
Azure Trusted Signing | Docker | jsign

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## Description
Are you distributing Windows Software outside of the Microsoft Store? For your users best experience and confidence, your applications should be codesigned.

This example shows how to codesign using Azure Trusted Signing.  
CodeSigning is using jsign in a Docker Container. This allows codesigning to be performed on Windows, macOS and Linux.
